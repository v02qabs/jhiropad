これは、jhiropadという、java言語で作られた
テキストエディタです。jcurses というツール
を導入しています。ので、コンソールや、GUI
環境と言った、２つの方法でjhiropadをお使い
いただけます。また、アプリ作成なんかに使う
バイナリをPCへ導入することにより、より早く
アプリ作成を行えることを目指して作成してい
きます。それでは、トップディレクトリで、
make 
mkae install
もしくは、
make all

と、コンソールで打ち、コンパイルもしくはあ
実行してください。
なお、パッケージの依存では、java se 8以降の
ランタイム、もしくは、java se jdk 8 で開発
しています。
そして、make パッケージをインストールする
事により、より一層、早くコンパイル＆実行
できます。
それでは、お楽しみください。
